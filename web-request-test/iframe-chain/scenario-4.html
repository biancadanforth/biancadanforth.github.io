Scenario 4: A chain of iframes, each loading a separate and distinct third party resource (an HTML document in this case).

Request 1
response.documentUrl biancadanforth.com
response.originUrl biancadanforth.com

Conclusion

The nested iframes' third party requests are picked up by webRequest.onResponseStarted, though the current capture code would discard any nested iframe requests as written now, since response.targetUrl is only different from response.documentUrl in the case of the first iframe.
